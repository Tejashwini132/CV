Project

Project Name:College Admision System
Frontend :Visual Basic 6.0
Backend :Oracle9i
Operation System :Windows XP

Descripiton:The main goal of the system is to automate the process carried out in the organization with imporved performance and realize the vision of paperless admission.
Some of the goals of the system are:

  • Manage large number of student details.
  • Manage all details of student who registered for the course.
  • Create student accounts and maintain the data effectively.
  • View all the details of students.
  • Reduce the work load in interview the student for selection.
  • Activities like updating,modification, deletion of records should be easier.
